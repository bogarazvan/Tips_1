LISTE

Tag-ul <OL>

se foloseşte pentru a defini o listă ordonată astfel:

Liste ordonate simple

  1. Primul element al listei
  2. Al doilea element al listei

Liste ordonate numerotate cu litere mici

  1. Primul element al listei
  2. Al doilea element al listei

Liste ordonate numerotate cu litere mari

  1. Primul element al listei
  2. Al doilea element al listei

Liste ordonate numerotate cu cifre romane mici

  1. Primul element al listei
  2. Al doilea element al listei

Liste ordonate numerotate cu cifre romane mari

  1. Primul element al listei
  2. Al doilea element al listei

Tag-ul <OL> poate avea un atribut START=6

se foloseşte pentru a stabili numerotarea intr-o listă ordonată astfel:

  1. Primul element al listei
  2. Al doilea element al listei

Tag-ul <UL>

se foloseşte pentru a defini o listă neordonată astfel:

Liste neordonate marcate cu disc

  • Primul element al listei
  • Al doilea element al listei

Liste neordonate marcate cu cerc

  • Primul element al listei
  • Al doilea element al listei

Liste neordonate marcate cu pătrat

  • Primul element al listei
  • Al doilea element al listei

Listă ordonată care cuprinde o listă neordonată

  1. Primul element
  2. Al doilea element
    • Primul subelement
    • Al doilea subelement
  3. Al treilea element
