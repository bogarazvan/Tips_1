Instrucţiuni de formatare fizică

Tag-ul <I>

se foloseşte pentru a impune un aspect inclinat spre stânga al textului cuprins între limitele sale

Tag-ul <B>

se foloseşte pentru a scrie textul dintre limitele sale îngroşat

Tag-ul <U>

se foloseşte pentru a scrie textul dintre limitele sale subliniat cu o linie

Tag-ul <BIG>

se foloseşte pentru a scrie textul dintre limitele sale mai mare ca restul

Tag-ul <STRIKE>

se foloseşte pentru a scrie textul dintre limitele sale tăiat cu o linie

Tag-ul <TT>

se foloseşte pentru a scrie textul dintre limitele sale monospaţiat care este specific maşinilor de scris vechi.

Tag-ul <SUP>

se foloseşte pentru a scrie o putere

Tag-ul <SUB>

se foloseşte pentru a scrie un indice

Instrucţiuni de formatare logică

Tag-ul <CITE>

se foloseşte pentru a scrie un citat dintr-o carte, revistă etc

Tag-ul <EM>

se foloseşte pentru a scrie un text evidenţiat

Tag-ul <SAMP>

se foloseşte pentru a pentru a da exemple de mesaje pe care calculatorul le poate transmite utilizatorilor. Fontul folosit este bold monospţiat

Tag-ul <STRONG>

se foloseşte pentru a scrie textul dintre limitele sale cu litere îngroşate

Tag-ul <VAR>

se foloseşte pentru a indica un nume de variabila.

Tag-ul <CODE>

Tag pentru descrierea programelor: Acest text este code

Diacritice sau caractere româneşti

Pentru precizarea unor caractere speciale sau a unor diacritice se poate folosi constructia &nume; sau &#numar;
Pentru diacritice este necesar ca in head să existe tag-ul:
<meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-2"> ca in această pagină web
