Definirea de legături către un alt document

Tag-ul <A> cu atributul HREF

se foloseşte pentru a defini o legătură către un nou document aflat în acelaşi director şi pe acelaşi disc local se face: <a href="Exemplu2.html">

Tag-ul <A> cu atributul HREF

Realizarea unei legături către o pagină situată într-un alt director aflat pe acelaşi disc local se face: <a href="/sokkit/site/formacnp.html">

Tag-ul <A> cu atributul HREF

Realizarea unei legături către o pagină web se face: <a href="http://www.google.com">

Tag-ul <A> cu atributul HREF căruia i se atribuie serviciul Internet mailto

permite lansarea în execuţie a unei aplicaţiei de expediere a mesajelor electronice <a href="mailto:popviorel@yahoo.com">
